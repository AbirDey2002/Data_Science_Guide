01 What is Statistical Testing?

A principled framework for making decisions under uncertainty — separating real signal from random noise in sampled data.

When: You can't observe the full population When: Deciding if a result is real or chance

Core Idea

You almost never observe everyone. You take a sample and draw conclusions about the bigger picture. The problem: samples are noisy. Statistical testing asks — is what I'm seeing real, or just random noise?

The Courtroom Analogy

Assume nothing unusual is happening (innocent until proven guilty = null hypothesis). Examine the evidence (data). Ask: how unlikely would this evidence be if nothing was actually going on? If very unlikely — reject the starting assumption.

02 Null Hypothesis

Two competing stories about the world. You never prove H1 directly — you make H0 look implausible.

When: Before every statistical test

The Two Hypotheses

H0 (Null) — "Nothing is happening. Any difference is chance."

H1 (Alternative) — "Something is genuinely happening."

Example

Testing if a new model recommends better than the old one:

  • H0: The two models perform the same. Any metric difference is random.
  • H1: The new model genuinely performs better.
You always build a case against H0 — never directly for H1.

03 P-Value

The probability of seeing data as extreme as yours (or more), assuming H0 is true.

When: Interpreting output of any statistical test

Interpretation

Small p → this data would be very unlikely if H0 were true → evidence against H0.
Large p → data like this is common even when nothing is happening → little evidence.

Alpha Threshold

Pre-select threshold alpha (α) before looking at data. Default α = 0.05.

  • p < α → reject H0 → statistically significant
  • p ≥ α → fail to reject H0
The p-value is NOT the probability that H0 is true. It's how surprising the data is, given H0. The most common and costly misinterpretation.

04 Type I & Type II Errors

Since you're deciding on incomplete data, you can be wrong in two distinct ways.

When: Designing experiments and setting thresholds
Error TypeWhat HappenedAlso CalledProbability
Type IRejected H0, but H0 was trueFalse Positive= α
Type IIFailed to reject H0, but H1 was trueFalse Negative= β

Power = 1 - β

Probability of correctly detecting a real effect. Target: 0.80 by convention.

The Tension

Stricter alpha → fewer false positives, but more false negatives. There is no free lunch. Like a smoke alarm: too sensitive → false alarms; not sensitive enough → misses real fires.

Setting α = 0.05 explicitly accepts a 5% false positive rate. This is a design choice, not an accident.

05 Statistical vs Practical Significance

A result can be statistically significant without being business-meaningful. These are completely different things.

When: Interpreting and reporting results

Why This Happens — Sample Size

With a large enough sample, tiny differences yield very small p-values. A 12.01% vs 12.00% CTR across 10M users might be statistically significant but practically irrelevant.

Effect Size

Tells you how big the difference is, independent of sample size. Always report alongside the p-value.

  • Cohen's d — difference in means in standard deviation units
  • Relative lift — % improvement over baseline
  • Cramér's V — effect size for Chi-Square
The reverse problem also exists: large, meaningful effects that miss significance because the sample was too small. That's a power problem, not proof of no effect.

06 Choosing the Right Test

Three questions point you to the right test almost every time. Wrong test = garbage results.

When: Before running any statistical test

  2. How many groups? One vs known value, two groups, or three or more
  3. Independent or paired? Different people per group, or same people measured twice

07 Independent T-Test

Tests whether the means of two independent groups are significantly different. The most common test in A/B testing.

When: Continuous data, 2 independent groups

T-Statistic — Signal to Noise Ratio

$$t = \frac{\bar{x}_A - \bar{x}_B}{\sqrt{\frac{s_A^2}{n_A} + \frac{s_B^2}{n_B}}}$$

Numerator = signal (group difference) | Denominator = noise (standard error)

Large t-statistic → difference is big relative to noise → strong evidence against H0.

Key Assumption

Data in each group should be roughly normally distributed. Usually fine for large samples due to the Central Limit Theorem.

from scipy import stats
import numpy as np

group_a = [42, 45, 41, 47, 43]
group_b = [35, 38, 33, 36, 34]

t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"T-statistic: {t_stat:.4f}")   # 7.2453
print(f"P-value: {p_value:.4f}")      # 0.0001

08 Paired T-Test

Same subjects measured twice. More sensitive than independent t-test because between-person variability cancels out.

When: Same people measured twice When: Before/after feature launch

Formula

$$t = \frac{\bar{d}}{S_d / \sqrt{n}}$$

$\bar{d}$ = mean of per-pair differences | $S_d$ = std dev of differences | $n$ = number of pairs

Step-by-Step

before = [12, 20, 15, 18, 10], after = [18, 26, 20, 22, 14]

  1. differences = [6, 6, 5, 4, 4]
  2. $\bar{d}$ = 25/5 = 5.0
  3. $S_d$ = 1.0
  4. t = 5.0 / (1.0 / √5) = 11.18
from scipy import stats

before = [12, 20, 15, 18, 10]
after  = [18, 26, 20, 22, 14]

t_stat, p_value = stats.ttest_rel(before, after)
print(f"T-statistic: {t_stat:.4f}")
print(f"P-value: {p_value:.4f}")
Running an independent t-test on paired data discards information and weakens your test. Always match your test to your data structure.

09 ANOVA — Analysis of Variance

Extension of the t-test for 3 or more groups. Tests all groups simultaneously, avoiding false positive inflation from multiple t-tests.

When: Continuous data, 3+ independent groups

F-Statistic

$$F = \frac{MS_{between}}{MS_{within}}$$

Large F → between-group variance dominates within-group noise → evidence against H0

Critical Limitation

ANOVA only tells you that a difference exists somewhere — not which groups differ. Use Tukey's HSD as a follow-up.

from scipy import stats

model_a = [12, 14, 11, 13, 12]
model_b = [20, 22, 19, 21, 20]
model_c = [15, 17, 14, 16, 15]

f_stat, p_value = stats.f_oneway(model_a, model_b, model_c)
print(f"F-statistic: {f_stat:.4f}")   # 91.6364
print(f"P-value: {p_value:.4f}")      # 0.0000

10 Tukey's HSD

Post-hoc test run after ANOVA. Identifies which specific pairs differ while controlling the family-wise false positive rate.

When: After ANOVA rejects H0, to find which pairs differ

How it Adjusts the Threshold

Uses the Studentized Range Distribution (q) instead of the t-distribution. This asks "how extreme is the largest difference among k groups?" — inherently stricter than a regular t critical value.

$$HSD = q_{\alpha,\,k,\,df} \times \sqrt{\frac{MSE}{n}}$$

$q$ = critical value from studentized range dist | $MSE$ = within-group variance from ANOVA | $n$ = obs per group

What is Controlled — FWER

The Family-Wise Error Rate is held at α = 0.05 across all comparisons together — not 5% per comparison. That's the guarantee multiple t-tests don't give you.

from statsmodels.stats.multicomp import pairwise_tukeyhsd

data   = model_a + model_b + model_c
groups = ['A']*5 + ['B']*5 + ['C']*5

result = pairwise_tukeyhsd(data, groups, alpha=0.05)
print(result)
# group1 group2 meandiff p-adj  lower   upper  reject
# A      B      8.0      0.001  6.296   9.704  True
# A      C      3.0      0.001  1.296   4.704  True
# B      C     -5.0      0.001 -6.704  -3.296  True
The lower/upper CI columns tell you the plausible range of the true difference — use these to assess practical significance, not just the reject column.

11 Chi-Square Test

Tests whether two categorical variables are independent by comparing observed counts to what would be expected under independence.

When: Categorical outcome (clicked/not, churned/retained) When: Comparing proportions across groups

Expected Count

$$E_{ij} = \frac{\text{Row Total}_i \times \text{Column Total}_j}{\text{Grand Total}}$$

Chi-Square Statistic

$$\chi^2 = \sum_{\text{all cells}} \frac{(O - E)^2}{E}$$

O = observed count | E = expected count | Larger gaps → larger χ² → smaller p-value

Degrees of Freedom

$$df = (rows - 1)(columns - 1)$$

Worked Example

Version A: 30 clicked, 70 didn't. Version B: 50 clicked, 50 didn't. Grand total = 200.

  1. Expected (both rows): 40 clicked, 60 didn't
  2. χ² = (30-40)²/40 + (70-60)²/60 + (50-40)²/40 + (50-60)²/60 = 2.5+1.67+2.5+1.67 = 8.08
  3. df = 1, p = 0.0045 → version influences click behavior
from scipy import stats

observed = [[30, 70],
            [50, 50]]

chi2_stat, p_value, dof, expected = stats.chi2_contingency(observed)
print(f"Chi2: {chi2_stat:.4f}")    # 8.0808
print(f"P-value: {p_value:.4f}")   # 0.0045
print(f"Expected:\n{expected}")
# [[40. 60.]
#  [40. 60.]]

12 Power Analysis & Sample Size

Ensures before running a test that you have enough data to detect a real effect if one exists. Prevents inconclusive experiments.

When: Designing any experiment or A/B test When: Before collecting data

The Core Problem

A failed test (p ≥ α) could mean: (a) there's truly no effect, or (b) your sample was too small. Power analysis removes this ambiguity before you start.

The Four Linked Quantities

QuantityTypical DefaultWhat it Controls
Alpha (α)0.05False positive rate
Power (1-β)0.80Ability to detect real effects
Effect size (MDE)Business decisionSmallest difference worth detecting
Sample size (n)Solve for thisData required per group

MDE — Minimum Detectable Effect

The smallest difference that would change a business decision. A business call, not statistical. Cohen's d conventions as fallback: 0.2 small, 0.5 medium, 0.8 large.

Sample Size Formula

$$n = \frac{2(z_{\alpha/2} + z_\beta)^2 \sigma^2}{\delta^2}$$

$z_{\alpha/2}$ = 1.96 for α=0.05 | $z_\beta$ = 0.84 for power=0.80 | $\sigma^2$ = variance | $\delta$ = MDE | Result is n per group

from statsmodels.stats.power import TTestIndPower
import numpy as np

analysis = TTestIndPower()

# Effect size = Cohen's d = MDE / std = 2 / 5 = 0.4
n = analysis.solve_power(
    effect_size = 2 / 5,
    alpha       = 0.05,
    power       = 0.80,
    alternative = 'two-sided'
)
print(f"Required n per group: {int(np.ceil(n))}")   # 99
Smaller MDE = dramatically more data. Detecting a 0.5 unit change requires ~16x more data than a 2 unit change. Set MDE based on what you'd actually act on.

13 One-Tailed vs Two-Tailed Tests

Determines whether you're testing for a difference in any direction (two-tailed) or only one specific direction (one-tailed).

When: Setting up your test before data collection
TypeQuestionCritical Value (α=0.05)P-value
Two-tailedDid anything change? (either direction)±1.96Full p
One-tailedDid it specifically improve?1.645p / 2
 
from scipy import stats

group_a = [42, 45, 41, 47, 43]
group_b = [35, 38, 33, 36, 34]

_, p_two = stats.ttest_ind(group_a, group_b, alternative='two-sided')
_, p_one = stats.ttest_ind(group_a, group_b, alternative='greater')

print(f"Two-tailed: {p_two:.4f}")   # 0.0001
print(f"One-tailed: {p_one:.4f}")   # exactly half
Choosing one-tailed after seeing the data direction is p-hacking — you artificially halved your p-value. Default to two-tailed unless you had a strong directional hypothesis before the experiment.

14 Confidence Intervals

A range of plausible values for the true effect. Richer than a p-value — tells you size and precision, not just yes/no.

When: Reporting results of any test When: Communicating uncertainty to stakeholders

Definition

A 95% CI means: if you repeated this experiment 100 times, ~95 of the resulting intervals would contain the true population value. It describes the process, not any single interval.

$$CI = (\bar{x}_A - \bar{x}_B) \pm t_{crit} \times SE$$

Relationship to P-Value

  • 95% CI does not contain zero → p < 0.05 → significant
  • 95% CI contains zero → p ≥ 0.05 → not significant

Why CI Beats P-Value for Communication

Same p-valueCIWhat it tells you
p = 0.04(0.01, 4.20)Real but uncertain — wide range
p = 0.04(1.80, 2.20)Real and precise — confident it's ~2
p = 0.04(0.01, 0.02)Real but practically meaningless
 
from scipy import stats
import numpy as np

group_a = [42, 45, 41, 47, 43]
group_b = [35, 38, 33, 36, 34]

mean_diff = np.mean(group_a) - np.mean(group_b)
se = np.sqrt(np.var(group_a, ddof=1)/len(group_a) +
             np.var(group_b, ddof=1)/len(group_b))
t_crit = stats.t.ppf(0.975, df=8)

lower = mean_diff - t_crit * se
upper = mean_diff + t_crit * se
print(f"95% CI: ({lower:.2f}, {upper:.2f})")   # (5.73, 11.07)

15 Communicating Results

Stakeholders don't care about p-values — they care about decisions. Lead with insight and action, not methodology.

When: Presenting findings to any stakeholder

The Communication Hierarchy

  1. What did we do? (one sentence)
  2. What did we find? (the number that matters)
  3. How confident are we? (significance + CI)
  4. What does it mean for the business? (the so what)
  5. What should we do? (the recommendation)

Bad vs Good

Bad: "We ran a t-test. T=7.24, p=0.0001, rejecting the null at α=0.05."

Good: "The new homepage increased session duration by 8.4 seconds — a 20% improvement. We're highly confident this is real (95% CI: 5.7s to 11.1s). We recommend full rollout."

Audience-Tailored Depth

AudienceWhat They Want
Product ManagerIs it real? How big? Ship or not?
EngineerConfidence level, edge cases
Exec / BusinessRevenue / cost impact, risk, recommendation
Fellow DSFull methodology, assumptions, limitations
Always anchor to baseline ("20% lift" not "8.4 second improvement") and always explicitly separate statistical significance from practical significance — stakeholders conflate them constantly.

Formula Reference Sheet

Every formula from the session, clean and in one place.

Independent T-Test Statistic

$$t = \frac{\bar{x}_A - \bar{x}_B}{\sqrt{\frac{s_A^2}{n_A} + \frac{s_B^2}{n_B}}}$$

Variables: $\bar{x}$ = group mean | $s^2$ = group variance | $n$ = group size

Used when: 2 independent groups, continuous data

Paired T-Test Statistic

$$t = \frac{\bar{d}}{S_d / \sqrt{n}}$$

Variables: $\bar{d}$ = mean of per-pair differences | $S_d$ = std dev of differences | $n$ = number of pairs

Used when: Same subjects measured twice

F-Statistic (ANOVA)

$$F = \frac{MS_{between}}{MS_{within}}$$

Variables: $MS_{between}$ = mean squared variance between groups | $MS_{within}$ = MSE (within-group variance)

Used when: 3+ independent groups, continuous data

Tukey's HSD Threshold

$$HSD = q_{\alpha,\,k,\,df} \times \sqrt{\frac{MSE}{n}}$$

Variables: $q$ = critical value from studentized range distribution | $MSE$ = within-group variance | $n$ = obs per group

Used when: Pairwise comparisons after ANOVA

Expected Count (Chi-Square)

$$E_{ij} = \frac{\text{Row Total}_i \times \text{Column Total}_j}{\text{Grand Total}}$$

Used when: Computing expected cell counts before Chi-Square test

Chi-Square Statistic

$$\chi^2 = \sum_{\text{all cells}} \frac{(O - E)^2}{E}$$

Variables: $O$ = observed count | $E$ = expected count

Used when: Testing independence of two categorical variables

Chi-Square Degrees of Freedom

$$df = (rows - 1)(columns - 1)$$

Used when: Looking up p-value for χ² statistic

Sample Size (Two-Sample T-Test)

$$n = \frac{2(z_{\alpha/2} + z_\beta)^2 \sigma^2}{\delta^2}$$

Variables: $z_{\alpha/2}$ = 1.96 for α=0.05 | $z_\beta$ = 0.84 for power=0.80 | $\sigma^2$ = variance | $\delta$ = MDE | Result is n per group

Used when: Determining required sample size before experiment

Confidence Interval

$$CI = (\bar{x}_A - \bar{x}_B) \pm t_{crit} \times SE$$

Variables: $t_{crit}$ = critical value for chosen confidence level | $SE$ = standard error of the difference

Used when: Reporting plausible range for the true effect

Standard Error (Difference of Means)

$$SE = \sqrt{\frac{s_A^2}{n_A} + \frac{s_B^2}{n_B}}$$

Variables: $s^2$ = variance of each group | $n$ = group size

Used when: Denominator of t-statistic and confidence intervals
